Feedback & Suggestions

10xTube is a free product, so there is no formal support service — but we genuinely read every message. Suggestions, feature requests, and bug reports are all welcome at 10xtube.support@gmail.com.

Quick Checks First

Most issues resolve with one of these:

  • Confirm that the extension is installed and enabled in Chrome.
  • Refresh any already-open YouTube tabs after installing or updating the extension.
  • Use a YouTube account where native YouTube Ask summaries are available.
  • Try a different public YouTube video if Summary is unavailable for the current video.
  • If using Listen with Azure, confirm that your Azure Speech key and region are configured in 10xTube's settings: open a video summary and click the gear icon next to Listen. See the Azure Speech key guide for setup steps, and How to Listen for narration and voice selection.

Known Limitations

YouTube Ask availability can vary by account, region, video, and YouTube UI rollout. 10xTube depends on YouTube page data and internal YouTube endpoints, so YouTube changes can affect behavior.

Reporting a Bug? Include

  • Chrome version.
  • 10xTube extension version.
  • The YouTube URL where the issue happened.
  • Whether the text summary, timestamp links, or Listen failed.
  • Any visible error message.
